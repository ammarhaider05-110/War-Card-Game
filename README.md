# War Card Game 

A digital version of the classic War card game built for iOS. 
Play against the CPU and be the first to reach 21 points to win!

## Gameplay
- Press **Deal** to flip cards for both you and the CPU
- The higher card wins the round and earns a point
- Ties give both players a point
- First to **21 points** wins the game

## Built With
- Swift
- SwiftUI
- Xcode

## Views
- **StartView** – Home screen with a start button
- **ContentView** – Main game screen with card dealing and score tracking
- **WinView** – Displayed when the player reaches 21 points
- **LoseView** – Displayed when the CPU reaches 21 points

## How to Run
1. Clone the repo
2. Open `War Card Game.xcodeproj` in Xcode
3. Select a simulator or device and hit Run
